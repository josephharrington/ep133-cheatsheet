how k.o. ii organizes music
project — 9 total, each is a song
└─ groups a/b/c/d — 12 pads each
└─ pattern — up to 99 bars, 99 patterns
└─ scene — one pattern per group, combined
└─ song — ordered list of up to 99 scenes
the commit loop
①
build a scene
record patterns across groups A–D. each group plays its own pattern simultaneously.
②
SHIFT + MAIN - commit
saves the current scene and creates the next one based on it. like "save as, keep going."
③
MAIN + ENTER - song editor
chain committed scenes into a song. each position holds one scene.
from-scratch workflow
▾
①
pick a project
1–5 come populated, 6–9 are empty by default - hold MAIN to check which one you're in
②
navigate to a scene
or let SHIFT+MAIN advance you automatically after each build
③
load sounds
select pad → browse with −/+. kicks 1–99, snares 100–199, etc.
④
set tempo & length
tap TEMPO for tap tempo. in RECORD (armed), −/+ changes bar count
⑤
record your pattern
4-beat count-in, then play pads. or RECORD+PLAY to start immediately
⑥
fix with step sequencer
step through with −/+, add notes with RECORD+PAD, remove with ERASE+PAD
⑦
add more groups
press A, B, C, or D to switch groups. each runs a separate pattern simultaneously
⑧
commit & vary the scene
saves this scene, clones it to the next - all groups carry over. modify any group from here (drop a hi-hat, add a fill, mute a pad), then commit again when ready
⑨
resample your output
captures the full mix of the current pattern as a new sample
⑩
build the song
SHIFT+A adds scenes in order. −/+ navigates positions. PLAY to hear it

easy to miss
  • Hold while turning any knob → slow/precise adjustment
  • Hold a pad in SOUND mode → see the sample name on screen
  • Umbrella icon lit up → undo available - press to undo
  • Hold while playing, then press GROUP pads → momentary per-group solo
  • Hold + or − during playback → temporarily nudge tempo up / down
  • In step mode: hold while pressing a pad longer → longer note duration
  • In SOUND mode, type a number (e.g. 1·0·3) + ENTER → jump directly to sample slot
  • Lock mode: hold on power-up → changes not auto-saved to disk
  • + → master compressor & sidechain settings
  • Tape stop (punch-in FX pad 2) affects MIDI clock output - useful for synced gear
  • Velocity/pressure off by default - in system settings: pad → vel → hi for soft-touch, pad → vel → lo for vigorous play
  • Mute groups: press , navigate to mute group sub-mode - pads in the same group cut each other off (great for hi-hats)
